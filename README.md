Excella Demo
=========

A testing/demo role.

Requirements
------------

Requirements are listed in the Pipfile, but to use that, install [Pipenv](https://docs.pipenv.org/)


Usage
-----

Run this with [Molecule](http://molecule.readthedocs.io/en/latest/) to grasp its full potential.

```
pipenv install
molecule test
```
